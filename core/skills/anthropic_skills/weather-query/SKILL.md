# 天气查询工具

## 功能说明
当用户询问某地的天气时，使用此工具查询实时天气信息。

## 使用方法
调用 weather_query 工具，参数为城市名称（中文或英文）。

## 返回格式
返回的天气信息包含：
- 城市名称
- 当前温度
- 天气状况（晴/阴/雨/雪等）
- 湿度百分比
- 空气质量指数
- 风力风向
- 穿衣建议

## 示例
用户问："北京今天天气怎么样？"
调用 weather_query(city="北京")

用户问："上海后天天气如何？"
调用 weather_query(city="上海")

## 注意事项
- 支持国内外主要城市
- 如果城市名有歧义，返回最可能的结果
- 部分小城市可能不支持
