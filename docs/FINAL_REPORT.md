### 1. 环境准备

```bash

pip install cryptography psutil

# 设置环境变量
export MIYA_ENCRYPTION_KEY="your-key-here"
```

### 2. 运行测试

```bash
# 运行E2E测试
python run_e2e.py

# 查看测试结果
cat test_results/e2e_test_results.json
```

### 3. 集成到现有系统

按照各模块的集成指南,将新功能集成到现有系统中:
- 在配置加载时启用缓存
- 在配置更新时使用批处理
- 在数据库操作时使用连接池
- 在存储敏感配置时启用加密
- 在API端点添加访问控制
- 在关键操作添加审计日志
- 在系统启动时启用监控

### 4. 启动系统

```bash
# 使用新的启动脚本
python run/main.py
```
